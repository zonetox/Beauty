# Admin Panel Functional Specification

## 1. Dashboard
- Total businesses
- Active memberships
- Pending orders
- Traffic overview
- Revenue charts

## 2. Business Management
- Approve / suspend businesses
- View full business profile
- Upgrade membership manually

## 3. Membership Packages
- Create / edit packages
- Toggle features per package
- Set duration and price

## 4. Orders & Payments
- View payment proof
- Approve / reject orders
- Order status tracking

## 5. Content Management
- Homepage section manager
- Blog management
- Landing page moderation

## 6. Reviews & Reports
- Approve / hide reviews
- Handle abuse reports

## 7. System Settings
- Site branding
- SEO defaults
- Email & payment config
