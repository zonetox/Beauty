# 🧠 MemoryOS Index

Hệ thống lưu trữ bộ nhớ dài hạn của dự án.

## Structure
- `/decisions/`: Lưu trữ các quyết định kiến trúc quan trọng (Why over What).
- `/lessons/`: Các bài học rút ra từ các lỗi đã sửa (Hậu Skillify).
- `/projects/`: Nhật ký tiến độ của từng module lớn.
- `/tasks/`: Lịch sử các tác vụ đã hoàn thành và cách thức thực hiện.

## Principle
*Intelligence increases proportional to memory recall.*
Mọi phiên làm việc phải kết thúc bằng việc cập nhật `decisions/` và `lessons/`.
