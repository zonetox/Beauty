# User Roles & Permissions Specification

## 1. Roles

### Guest
- View directory
- Search businesses
- View landing pages
- Read blog

### Business Member
- Edit business profile
- Manage landing page content
- Manage media, services, blog
- View analytics
- Manage membership & orders

### Staff (Sub-user)
- Edit landing page content
- Edit blog
- No access to billing or membership

### Admin Roles
- Super Admin: full access
- Admin: user, content, orders
- Editor: blog only
- Moderator: reviews & reports

## 2. Permission Principles
- Role-based access control (RBAC)
- Membership package unlocks features
- Admin can override permissions
